How to Implement

1. Include the Script

<script src="https://your-domain.com/embed.js"></script>

2. Add the Widget

<div data-jobsearch-widget="profile" data-theme="modern"></div>

3. Listen for Events (Optional)

// Listen for user navigation window.addEventListener('jobsearch:navigate', (event) => { const path = event.detail.path; // Handle navigation to your app pages window.location.href = path; }); // Listen for user logout window.addEventListener('jobsearch:logout', () => { // Handle user logout in your application console.log('User logged out'); });

Features

  • Automatic Authentication Detection: Shows appropriate UI based on user login state
  • Multiple Themes: Choose from modern, classic, or minimal designs
  • Responsive Design: Works perfectly on desktop and mobile
  • Custom Events: Integration hooks for navigation and logout
  • Secure: Uses localStorage for user session management
  • Lightweight: Minimal footprint with no external dependencies
